Background of Inside elbow punch

Using the inside of the elbow creates many opportunities in sparring and fighting. This video shows some techniques into which an inside elbow punch can be integrated.

  1. Hitting the chest
  2. Hitting the chin
  3. Hitting the back of the neck with the inside elbow
  4. Using the inside elbow to prepare for Kubi nage - Neck throw
  5. Using the inside elbow punch to make Kata guruma - Shoulder wheel work
