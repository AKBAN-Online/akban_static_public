Uchi mawashi dori - Single hand release

By Yossi Sheriff

Video of Uchi mawashi dori - Single hand release


Uchi mawashi dori is a Ninjutsu technique a single hand te hodoki release and is a quick and effective one to use- even against stronger opponents. The release doesn't require a lot of strength to be effective but does require proper technique. The user of the release moves to the opponent's side- a better position from which to strike him.

Description of Uchi mawashi dori - Single hand release

When performing this technique the user moves forewards and sideways (to the same direction of the held hand) while moving the elbow towards the opponent's elbow. The hand is released automatically if the elbows are near enough as the opponent will not have time to reposition himself.
