Sunday in Jerusalem dojo - Using the jig with second timing

This Sunday we will continue practicing the second timing sequences while using the Jig sabaki stepping method. Using this step we will explore Te Makura, Osoto gake, Fudo Ken and Ura gyaku using the Jig as a timing rather then distancing combat method.
